# docs/

What you produce. Strategy, PRDs, research, user tracking, GTM. This is institutional memory — treat existing docs as the foundation, not optional context.

| Folder | What goes here |
|---|---|
| `strategy/` | Strategy docs, execution plans, positioning |
| `narrative/` | Messaging frameworks, narrative docs |
| `gtm/` | Go-to-market plans, launch docs |
| `market-research/` | Competitor analyses, market sizing, landscape scans |
| `product/` | PRDs, specs, roadmap |
| `users/` | User tracking, relationships, follow-ups, processed feedback |

**Before starting any research or writing:** search this folder first. Build on what exists. Only invoke new research if there are genuine gaps.
