# projects/

Personal projects. Separate from work output in `docs/`. Anything you're building, writing, or growing outside your primary job lives here.

Suggested structure:

```
projects/
├── growsocial/       # Social content engine (X, LinkedIn, blog)
│   ├── planning/     # Experience bank, theses, quality gates
│   ├── x/            # X/Twitter drafts, engagement log
│   └── li/           # LinkedIn drafts
└── {{YOUR_PROJECT}}/ # Whatever you're building
```
