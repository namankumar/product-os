# context/

Current state of the world. Not exploration, not WIP — ground truth that Claude reads before every session.

| File | What goes here |
|---|---|
| `current-context.md` | Decision-making context that doesn't live elsewhere. What's changing, what's uncertain, what matters right now. |
| `okr-status.md` | OKR targets vs actuals. Updated weekly. |
| `narrative-map.md` | Active narratives: what to push, what to avoid, gaps. |
| `cadences.md` | Recurring events: standups, all-hands, OKR check-ins, sprint cycles. |
| `artifacts.md` | Registry of key external docs (Google Docs, Sheets, Slides) with IDs and purpose. |
| `org-dynamics.md` | How decisions get made, where tensions live, who has influence. |
| `room/` | Group dynamics analysis. One file per group (team, investors, partners). |

**Rule:** Claude reads these files before answering questions or starting research. Keep them current or the AI answers get stale.
